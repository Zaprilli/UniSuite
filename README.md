# UniSuite
# 🎓 UniSuite - Student Lifestyle Platform

Монгол оюутан, залуусын амьдралын хэв маяг, уулзалт, баяр ёслолын төлөвлөлтийг хөнгөвчлөх автоматжуулсан вэб системүүдийн багц.

---

## 🚀 Төслийн Бүтэц (Apps)

1. **👩‍❤️‍👨 UniSuite - Bolzoo (`/apps/bolzoo`)**
   - Кофе шоп, музей, арт галерей, кино, хоол, пикник зэрэг болзооны урилга үүсгэх вэб.
2. **💝 UniSuite - Bayar (`/apps/bayar`)**
   - Валентин, Мартын 8, Шинэ жилийн тусгай баярын вэб загварууд болон бэлэгний Wishlist.
3. **🍻 UniSuite - Uulzi (`/apps/uulzi`)**
   - Заал, PC, караоке, уулзалт, бильярд, боулинг тоглохоор найзуудаараа цаг товлож, Vote хийх систем.
4. **🎂 UniSuite - HBD (`/apps/hbd`)**
   - Төрсөн өдрийн дижитал урилга болон RSVP (Ирэх хүмүүсийн бүртгэл).

## 🛠 Ашиглах Технологиуд (Stack)
- **Frontend:** Next.js (React), TailwindCSS, Shadcn/ui
- **Backend/Database:** Supabase эсвэл Firebase (Оюутны төсөлд үнэгүй бөгөөд маш хурдан)
- **Deployment:** Vercel

## 💻 Орон нутагт асаах (Getting Started)

Эхлээд төслийг өөрийн компьютер дээрээ татаж авна:
\`\`\`bash
git clone https://github.com/[чиний-github-нэр]/unisuite.git
cd unisuite
\`\`\`

Шаардлагатай package-уудыг суулгах:
\`\`\`bash
npm install  # эсвэл pnpm install / yarn
\`\`\`

Төслийг ажиллуулах:
\`\`\`bash
npm run dev
\`\`\`
